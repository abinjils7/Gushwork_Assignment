# Mangalam HDPE Pipes - Frontend UI Assignment

A responsive, pixel-perfect frontend implementation of the Mangalam HDPE Pipes product landing page, built as part of the Gushwork UI assignment.

## 🚀 Features & Technical Highlights

This project was built entirely using **Vanilla HTML5, CSS3, and JavaScript** with strict adherence to modern performance and accessibility standards. Zero external libraries or frameworks were used.

- **Pixel-Perfect Responsive Design**: Flawless breakdown across Desktop (1200px+), Tablet (768px+), and Mobile (<767px) viewports using modern CSS Grid and Flexbox architectures.
- **Hardware-Accelerated Interactivity**: 
  - Smooth **Sticky Header** controlled via `requestAnimationFrame` and `IntersectionObserver`.
  - Custom **Image Zoom Floating Preview** built from scratch utilizing cursor coordinate math, completely bypassing heavy zooming plugins.
  - Interactive **Manufacturing Tabs** and an extendable **FAQ Accordion** with seamless CSS transitions.
- **Advanced Frontend Polish**:
  - Premium **3D Mouse Tracking Tilt** effects applied to cards for a modern, tactile glassmorphism feel.
  - Performance-optimized **Scroll Reveal Animations** built natively with the `IntersectionObserver` API.
  - Fully bespoke UI elements including a custom WebKit scrollbar and custom accessible `:focus-visible` states.
- **Production-Ready Performance**:
  - `will-change` properties applied to complex animations to ensure smooth 60fps rendering without latency or lag.
  - LCP Image preloading and optimized DOM loading strategies.
  - Extensive use of semantic HTML5 tagging and Open Graph SEO meta properties.

## 🛠 Setup & Installation

Because this project is built with vanilla web technologies, no build step or package installation (`npm install`) is required. 

1. Clone the repository:
   ```bash
   git clone https://github.com/abinjils7/Gushwork_Assignment.git
   ```
2. Open `index.html` directly in any modern browser, or use a local development server (like VSCode Live Server) for the best experience.
